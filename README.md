# MouseEvent
## 목적
- 마우스 이벤트를 이용하여 
## 기능

- 마우스 커서가 이미지에 들어갔을 때 
    - 이미지를 확대하여 보여줄 컨테이너 표시
    - 마우스 커서를 따라 이미지가 확대되는 범위 표시
- 마우스가 이미지 위에서 움직일 때
    - 마우스 커서를 따라 이미지 확대 위치 변경
    
- 마우스가 이미지에서 나갔을 때
    - 확대 이미지가 나타난 컨테이너를 숨김
    - 마우스 커서를 따라 이미지가 확대되는 범위 숨김
    
- 이미지를 드래그 할 때
    - 추가 이미지가 있는 경우 드래그 방향의 이미지로 변경
